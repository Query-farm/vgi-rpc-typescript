## Features

<CardGrid>
  <Card title="Three Method Types" icon="list-format">
    Unary (request-response), producer (server-streaming), and exchange (bidirectional-streaming) methods.
  </Card>
  <Card title="Apache Arrow IPC" icon="rocket">
    Efficient columnar serialization compatible with the Python vgi-rpc framework. No JSON overhead.
  </Card>
  <Card title="Schema Shorthand" icon="pencil">
    Declare schemas with `{ name: str, count: int }` instead of manual `Schema`/`Field` construction.
  </Card>
  <Card title="HTTP Transport" icon="globe">
    Serve methods over HTTP with stateless streaming via HMAC-signed state tokens. Works with Bun, Deno, and Cloudflare Workers.
  </Card>
  <Card title="Fluent Protocol Builder" icon="setting">
    Chain `.unary()`, `.producer()`, `.exchange()` calls to define your service.
  </Card>
  <Card title="Runtime Introspection" icon="magnifier">
    Opt-in `__describe__` method for dynamic service discovery via the CLI.
  </Card>
</CardGrid>
